How to Install / Uninstall the game?

Astroseries - How to Install

1) Copy the "Astroseries" folder to your hard disk. (Just drag the folder to where you want it on your hard disk and wait until copying is completed)

2) Once on your hard disk, run freecraft.exe to begin the game.


To Uninstall

Just delete the "Astroseries" folder by sending it to the trash bin. Once you empty the trash bin, the game is completely gone from your computer.



What is Astroseries?

Astroseries, or Astroseries RTS, as it is often known as, is a Realtime Strategy game set in a futuristic era, featuring a conflict between several interstellar nations.


How do I start Astroseries?


Double click on the freecraft.exe (or corresponding icon if on Linux or other operating system) icon present in this same directory.


The game does not start. What do I do?

On some computers the game takes quite longer to load. Wait a bit. If, however, you suspect the game will not run, try reinstalling the game.


I was playing a campaign and saved a game. When I came back and tried to load it, the game crashed. What now?

Please refer to the next question. It gives you the chance to skip levels in a campaign. Thats the best fix we have found to this problem so far. This problem is expected to be solved over time.


Can I skip levels in a campaign?
  Yes, you can! While in the name of fun we recommend you to play all levels, there might be situations in which you will want to skip levels in a campaign. You might want to skip a level if:

  • You are stuck in a level and cannot go on (for some reason).
  • You saved a game but when you try to load it, the game crashes.

This version of Astroseries includes pre-saved games. You may want to Load any of them from human01.sav.gz to human16.sav.gz for the Imperial Union, or nagoran01.sav.gz to human16.sav.gz for the Nagoran Empire.

Besides the one described above, there are other two methods to skip levels.

Method 1

1) During gameplay, press ENTER (so you can type a message)

2) Type (lowercase and all like this) the following: unite the clans

3) Press ENTER again.

The current level should end, in victory for you. You will be taken to the next level in a campaign. If you are using this because a saved game became corrupted and would not load, repeat for as many levels until you get to where you were :-)


Method 2

  There is a way of running individual levels. To play any given level any time, follow these steps:

1) Run stratagus.exe (or the corresponding file if you are on Linux or other operating system)
2) From the menu, choose "Single Player Game".
3) A couple of options show up. Click on "Select Scenario". A dialog box show up.
4) In "Type", select Stratagus scenario (cm).
5) Then right below "Map size", you see a folder with an up arrow that says "maps" next to it. Click it.
6) Click on campaigns.
7) Now you are given two choices: human or nagoran   Click on the one you want. NOTE: human is the Imperial Union.
8) A list of maps show up, from level01a.cm to level16a.cm
9) Suppose you want to play level 9 from the human race, then click on level09h.cm
10) The dialog box closes. Map is ready. Now click on "Start Game".
11) Enjoy!



The game behaves in a strange way

If the game behaves in a strange way, like there was sound before and now there isn't, or the mouse cursors are not those supposed to be, or any other erratic behavior, what you should do is save the game, exit the program, and run again. If when saving the game the keyboard does not respond (sometimes happens), choose default.sav.gz with the mouse and save there (that's what this file is for). The next time you run the game, all those strange symptoms should be gone.



When attacking planetary defenses, my units attack the planet and NOT the planetary defenses. How do I do?

If your units attack the planet instead of the planetary defenses, make them STOP (by clicking on their STOP button) so they stop the attack. They will pick and attack the correct target automatically.


When using the Map Editor, how do I delete units?

Move the mouse cursor over the unit you want to remove. Then press the DELETE key on your keyboard.

What is Stratagus?

Stratagus is a gaming engine that lets you create your own RTS (Real Time Strategy) games, similar to WarCraft I / WarCraft II, StarCraft, Red Alert, Ages of Empires, etc. So far, it can only create RTS games in 2D, but it is still very cool nonetheless. Mastering the engine takes its time, as you at least need to have some scripting experience of any type to understand what it is all about. And making a game generally requires certain conmitment on the part of the creator, as it generally takes a good while. A typical RTS game can easily take between two to four months, if not more, to complete. Astroseries took about seven months to make, and a good while more to improve it over time.
